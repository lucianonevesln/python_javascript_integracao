# Projeto: Integração Python-JavaScript-HTML

## Objetivo:

### Criar uma aplicação que faça com que as tecnologias de Python, JavaScript e HTML troquem informações entre si.

## Como Funciona:

### O usuário insere o nome completo, e-mail e ao clicar em Enter, esses dados são enviados ao servidor por meio de JQuery/AJAX, que retorna uma resposta em inputs no front-end.

## Tecnologias Usadas:

### - Python;

### - JavaScript;

### - JQuery;

### - AJAX;

### - HTML;
